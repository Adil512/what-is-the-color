## ğŸ¯ Getting Started

### Prerequisites

All you need is a modern web browser that supports:
- HTML5
- CSS3
- ES6 JavaScript
- Canvas API

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Adil512/what-is-the-color.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd what-is-the-color
   ```

3. **Open in your browser**
   ```bash
   # Simply open index.html in your browser
   # Or use a local server (recommended)
   python -m http.server 8000
   # Or with Node.js
   npx http-server
   ```

4. **Visit the application**
   - Open `http://localhost:8000` in your browser

### Alternative: Direct Use

Simply download the repository and open `index.html` in any modern web browser. No build process or dependencies required!

## ğŸ“š Usage

### Basic Color Selection

1. **Use the Color Picker**: Click on the color picker input to select any color
2. **Generate Random Colors**: Click the "Random Color" button for inspiration
3. **View Color Information**: See HEX, RGB, HSL, and CMYK values instantly

### Copying Color Values

- Click the ğŸ“‹ icon next to any color format to copy it to your clipboard
- Use the copied values in your CSS, design tools, or anywhere you need

### Exploring Color Shades

- Automatically generated color variations appear below the main color
- Click any shade to make it the active color
- Perfect for creating color palettes

### Extracting Colors from Images

1. Click the "Upload Image" button
2. Select any image from your device
3. View the top 10 dominant colors extracted from the image
4. Click any extracted color to explore it further

### Keyboard Shortcuts

- `Ctrl/Cmd + R` - Generate random color
